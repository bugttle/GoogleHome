# Google Home
Google Home applications on Node.js.

## Applications
### LinkStation
- Login to admin web page and Send request to shutdown.
- Get power status.

### PlayStation4
- Wake.
- Do standby.
- Launch application.
    - torne


## Install
- node packages
~~~shell
$ npm install
~~~

- ruby gems
~~~shell
$ gem install net-ping nokogiri
~~~

- launch
~~~shell
$ node index.js
~~~
